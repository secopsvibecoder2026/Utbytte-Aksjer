Bakgrunn og formål

exday.no startet som et privat verktøy for å holde oversikt over ex-datoer og utbyttebetalinger fra norske aksjer. Som utbytteinvestor på Oslo Børs er det tidkrevende å holde styr på hvilke selskaper som betaler utbytte, når ex-datoen er, og hva den faktiske direkteavkastningen er til enhver tid. Det fantes gode verktøy internasjonalt, men lite skreddersydd for norske investorer på Oslo Børs.

Over tid vokste prosjektet til en fullverdig PWA (Progressive Web App) med porteføljesporing, utbyttekalkulator, sektoranalyse og personlig IRR-beregning. Siden drives som et hobbyprosjekt på fritiden og er tilgjengelig gratis for alle.

Ingen innlogging, ingen abonnement. Alle data lagres lokalt i nettleseren din. exday.no sender ikke personlig investorinformasjon til noen server.

Et AI-drevet hobbyprosjekt

exday.no er bygget med betydelig hjelp fra kunstig intelligens. Fra kodebase til innholdsproduksjon benyttes moderne AI-verktøy gjennom hele utviklingsprosessen. Det gjør det mulig for én person å bygge og vedlikeholde et prosjekt av denne skalaen på fritiden.

AI brukes konkret til å:

  • Generere beskrivelser for 191 aksjer — basert på faktiske nøkkeltall og selskapsinformasjon, ikke generiske maler
  • Skrive og forbedre kode — JavaScript, Python og HTML/CSS for front-end og datapipeline
  • Analysere datakvalitet — oppdage anomalier i Yahoo Finance-data, som feil annualisering eller dobbeltbokføring av utbytter
  • Produsere artikler og forklaringer — innhold som hjelper investorer å forstå begreper som ex-dato, skatt og yield

Prosjektet er åpent om at AI er en sentral del av produksjonen. Alle tall, datoer og finansielle data er hentet fra offentlige kilder og valideres automatisk — AI brukes til struktur og språk, ikke til å finne opp fakta.

Teknologi og infrastruktur

Hele tjenesten kjører som statiske filer på GitHub Pages uten noen server-side kjøretid. Det betyr ingen driftsutgifter, høy oppetid og rask lastetid. Alle beregninger skjer i nettleseren din.

Teknologistakk:

GitHub Pages Vanilla JavaScript Tailwind CSS v4 Python 3.12 GitHub Actions Yahoo Finance API Euronext CSV Service Worker PWA

Kildekoden er åpen og tilgjengelig på GitHub. Det brukes ingen tredjeparts JavaScript-rammeverk — alt er skrevet i ren («vanilla») JavaScript for å holde lastetiden lav og avhengighetene minimale.

Automatiserte oppdateringer

Data hentes og oppdateres automatisk via to GitHub Actions-jobber:

  • Daglig pipeline (kl. 08:00 CET, hverdager) — henter utbytteinformasjon, ex-datoer, historiske utbytter og nøkkeltall for alle 191 aksjer via Yahoo Finance. Genererer individuelle SEO-sider per aksje og oppdaterer datagrunnlaget.
  • Kurspriser hvert 15. minutt (08:00–17:00 CET, hverdager) — henter siste kurs fra Euronext og Oslo Børs slik at direkteavkastningen alltid beregnes mot oppdatert kurs.

Automatiseringen gjør at siden holder seg oppdatert uten manuelt arbeid, selv om ingen er innom siden på dager.

Datakilder og nøyaktighet

Finansielle data hentes primært fra Yahoo Finance og Euronext. Begge kildene er gratis og offentlig tilgjengelige, men de har begrensninger som er viktige å kjenne til:

  • Forsinkelse: Kurspriser kan ha inntil 15 minutters forsinkelse i børsåpningstiden.
  • Annualisering: Yahoo Finance annualiserer utbytte basert på siste betaling multiplisert med betalingsfrekvens. For selskaper med uregelmessige eller varierende utbytter kan dette gi misvisende høy eller lav yield.
  • Valuta: Aksjer på Oslo Børs handles i NOK, men noen selskaper rapporterer i USD. Historiske utbytteverdier kan vise avvik sammenlignet med andre kilder på grunn av valutakursendringer.
  • Manglende data: Noen selskaper har ufullstendig historikk i Yahoo Finance, særlig nynoterte selskaper eller selskaper med liten internasjonal dekning.
All data på exday.no bør brukes som et utgangspunkt for videre research — ikke som eneste kilde for investeringsbeslutninger. Verifiser alltid mot Oslo Børs, selskapets årsrapport eller megler.

exday.no kjører automatisk datakvalitetskontroll etter hver henting. Selskaper med avvikende yield-beregninger, suspekte historiske verdier eller inkonsistente nøkkeltall flagges internt og kan vises med advarsel.

Hva kan du bruke exday.no til?

Tjenesten dekker hele livssyklusen til en utbytteinvestor på Oslo Børs:

  • Utbyttekalender — se hvilke aksjer som har ex-dato de neste ukene og månedene, sortert etter dato.
  • Aksjesøk og filtrering — finn aksjer etter sektor, yield, frekvens (månedlig, kvartalsvis, halvårlig, årlig), utbyttevekst og konsistens.
  • Porteføljeoppfølging — legg inn kjøp og salg med dato og kurs. Beregn kostbasis (FIFO), realisert og urealisert gevinst, internrente (IRR) og tidsvektet avkastning (TWR).
  • Skatteberegning — estimat på utbytteskatt etter skjermingsfradrag med gjeldende effektiv skattesats.
  • Sektorrebalansering — sett ønsket sektorvekting og se hvor mye du bør kjøpe eller selge for å nå målvektene.
  • Utbyttekalkulator — simuler fremtidig utbyttestrøm med reinvestering og forventet vekst.
  • Dividendscore — et sammensatt poengsystem som veier yield, konsistens, vekst og bæreevne for rask sammenligning.

Alt lagres lokalt i nettleseren din. Du kan eksportere porteføljen som CSV eller JSON, og importere den igjen på en annen enhet via QR-kode eller fil.

Personvern og cookies

exday.no respekterer personvernet ditt. Siden bruker Google Analytics for å forstå hvilke sider og funksjoner som er mest nyttige, men dette krever ditt samtykke. Første gang du besøker siden, blir du spurt om du aksepterer analytiske informasjonskapsler.

Din porteføljedata, favoritter, watchlister og innstillinger lagres utelukkende i nettleserens localStorage — de sendes aldri til en server og er kun tilgjengelig på din enhet. exday.no har ingen brukerdatabase og lagrer ingen personopplysninger.

Ansvarsfraskrivelse

exday.no er ikke et verdipapirforetak og tilbyr ikke finansiell rådgivning. Innholdet på siden er kun til informasjonsformål og skal ikke tolkes som råd om kjøp eller salg av verdipapirer.

Historisk utbyttedata er ingen garanti for fremtidig utbytte. Selskaper kan redusere, utsette eller stanse utbyttet uten forvarsel. Alle investeringsbeslutninger bør baseres på grundig egen analyse og eventuelt rådgivning fra en autorisert finansrådgiver.

Data kan inneholde feil eller forsinkelser. exday.no tar ikke ansvar for tap som oppstår som følge av informasjon hentet fra denne siden.
