1. Om denne siden

exday.no er en informasjonstjeneste som viser oversikt over norske utbytteaksjer notert på Oslo Børs. Siden samler ikke inn personopplysninger på egne servere, men benytter tredjepartstjenester fra Google som kan sette informasjonskapsler på din enhet – dersom du samtykker til det.

Denne siden er i samsvar med GDPR (General Data Protection Regulation) og norsk personvernlovgivning forvaltet av Datatilsynet.

2. Informasjonskapsler

Vi bruker kun sporings-informasjonskapsler dersom du aktivt samtykker til det ved første besøk. Du kan trekke tilbake samtykket ditt når som helst i seksjon 5 nedenfor.

Nødvendige (alltid aktive)

cookie_consent — lagrer ditt samtykkevalg. tema — mørkt eller lyst utseende. Ingen av disse sendes til eksterne servere.

Analyse (krever samtykke)

_ga, _gid — Google Analytics 4. Brukes til anonymisert statistikk om sidetrafikk (sidevisninger, enhetstype, omtrentlig geografisk plassering). IP-adresser anonymiseres. Varighet: 2 år / 24 timer.

Annonsering (krever samtykke)

Google AdSense — leverer annonser via Googles annonsenettverk. Google kan bruke informasjonskapsler til å vise relevante annonser basert på dine interesser. Se Googles annonsepolitikk for detaljer.

3. Google Analytics

Vi bruker Google Analytics 4 for å forstå hvordan besøkende bruker siden. Dataene er aggregerte og anonymiserte — vi ser ikke hvem du er, bare mønstre som hvilke sider som er populære og hvilke enheter som brukes.

Data behandles av Google LLC (USA) under EUs standard databehandlervilkår. Les mer i Googles personvernerklæring.

4. Lokal datalagring

exday.no lagrer brukerpreferanser og appdata direkte i nettleseren din (localStorage og Cache API). Ingen av disse dataene sendes til exday.no sine servere eller deles med tredjeparter — de finnes kun på din enhet.

Nøkkel Innhold Formål
fav_aksjer Liste over tickers du har stjernemerket Huskeliste / favoritter
pf_beholdning Tickers og antall aksjer i kalkulatoren Porteføljeberegning
notif_aksjer Tickers du har aktivert ex-dato-varsler for Varselinnstillinger
notif_vist_* Hvilke varsler som allerede er vist Hindre duplikate varsler
Cache API Statiske filer og siste aksjedata Offline-støtte (PWA)

Du kan slette all lokal data via nettleserens innstillinger: Innstillinger → Personvern → Nettleserdata → Nettstedsdata → exday.no.

5. Dine rettigheter

Etter GDPR har du rett til å:

  • — Få innsyn i hvilke opplysninger som behandles om deg
  • — Kreve retting eller sletting av opplysninger
  • — Protestere mot behandling og be om begrensning
  • — Trekke tilbake samtykke når som helst uten at det påvirker lovligheten av tidligere behandling
  • — Klage til Datatilsynet dersom du mener personvernet ditt er krenket

Siden lagrer ingen personopplysninger på egne servere. For henvendelser om data hos Google, bruk Googles personverndashboard.

6. Administrer samtykke

Du kan når som helst endre eller trekke tilbake samtykket ditt her. Endringen trer i kraft ved neste sideinnlasting.

Kurs og utbyttedata hentes fra Yahoo Finance og Euronext. Oppdateres daglig på børsdager.

exday.no er ikke et verdipapirforetak og tilbyr ikke finansiell rådgivning. Data kan inneholde feil eller forsinkelser — verifiser alltid mot Oslo Børs eller selskapets egne rapporter. Historisk utbytte er ingen garanti for fremtidig utbytte. Gjør alltid din egen analyse før du tar investeringsbeslutninger.
